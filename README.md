Xの投稿内容をBlueskyにクロスポストするUserScript
素人がチャットAIで作ったのでまともに動かないと思います
一応、注意事項は以下

- 動作は無保証、使う場合は自己責任でお願いします
- MacのSafariとTampermonkey互換の機能拡張Stayでしか動作確認してません
- 元ネタの https://github.com/59de44955ebd/twitter-to-bsky と比べると、Mastodon処理がオミットされ、投稿内容のハッシュタグとURLがハイパーリンク化されます
- 画像を添付すると、XにはポストされますがBlueskyにはポストされません
- 動画の添付はテストしてません
- GitHubの作法を知らないので、要望やリクエストをもらっても反映方法や修正の仕方すら分かりません

使い方

- Safariに機能拡張Stayを組み込んで有効化
- Stayにこのスクリプトを読み込ませる
- Xの投稿画面を表示
- 青い変なアイコンをクリックしてBlueskyのハンドルとアプリパスワードを入力してsave
- 変な青いアイコンをもう一回クリックしてcrosspost to Blueskyをチェックしてsave
- 何か書いてXにポストすると、上手くいけばBlueskyに内容がクロスポストされます
