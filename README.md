X (旧 Twitter) での投稿時に、指定したBlueskyアカウントへ同時にクロスポストするためのTampermonkey/Greasemonkey用ユーザースクリプトです。
Buffer 連携機能も搭載しており、モバイル/デスクトップ環境に応じて最適な投稿ウィンドウを開くことも可能です（が、十分な実装ではありません。特に動画投稿）。

✨ 主な機能
 - X の投稿を Bluesky にクロスポスト: X のポスト編集画面内に専用チェックボックスを追加し、チェックを入れるだけで Bluesky へ同時投稿 （現状では、テキスト、ハッシュタグ、URLリンク、引用ポストに対応）。
 - Buffer ポップアップ連携: 左側のナビゲーションメニューの投稿ボタンやモバイル環境の投稿ボタンを押した際に、X のポスト編集画面ではなく Buffer の投稿ウィンドウを自動で開くオプションを提供。
 - シンプルな設定: X の画面に Bluesky 設定アイコンを追加し、PDS URL、ハンドル、App Password を簡単に設定・保存可能。

📥 インストール方法
1. ユーザー スクリプトマネージャーをインストール
このスクリプトを実行するには、以下のいずれかのブラウザ拡張機能が必要です。Mac/iPad/iPhone の Safari と Tampermonkey の組合せでテストしましたが、他でも動くかもしれません。

Tampermonkey （推奨）や各種の互換機能拡張
Greasemonkey
Violentmonkey
Stay など

2. スクリプトをインストール

注意：うまく動かない時は Tampermonkey を上級者設定にして、セキュリティの @connect 文を確認 項目を緩めたり、実験的項目の厳格モードを無効にしてみてください。

⚙️ 設定手順
1. X (旧 Twitter) のタイムラインを開きます。
2. 画面左側、またはモバイル環境の投稿モーダル内に追加された Bluesky の設定アイコン (青い菱形のアイコン) をクリックします。
3. 表示された設定パネルに、以下の情報を入力し、「Save」ボタンをクリックして保存します。
  - PDS URL: https://bsky.social (通常はデフォルトのままでOK)
  - Handle: あなたの Bluesky ハンドル (xxxx.bsky.social)
  - App Password: Bluesky の設定で生成したアプリケーションパスワード (xxxx-xxxx-xxxx-xxxx 形式)
  - （オプション）Buffer 連携を有効化する場合: 「Open Buffer post window」にチェックを入れます。

📝 使い方
- Bluesky クロスポスト
  1. X の投稿編集画面を開きます。
  2. 投稿編集画面に追加された 「Bluesky」チェックボックス にチェックを入れます。
  3. 通常通り投稿ボタンをクリックします。
  4. X への投稿と同時に、Bluesky へのクロスポストが実行されます。投稿成功・失敗は通知で確認できます。

- Buffer 連携機能
  1. 設定パネルで「Open Buffer post window」にチェックを入れ、保存します。
  2. X の画面左側ナビゲーションにある「ポストする」ボタン（またはモバイル環境の右下の投稿ボタン）をクリックします。
  3. X の投稿編集画面の代わりに、Buffer の投稿ウィンドウが新しいポップアップまたはタブで開きます。画像や動画を添付して投稿することが多いなら、こちらが便利かもしれません。

⚠️ 注意事項
セキュリティのため、App Password は必ず Bluesky の公式設定画面で生成したものを使用してください。
