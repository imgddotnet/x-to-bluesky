X（旧 Twitter）での投稿時に、Blueskyへ同時にクロスポストするためのTampermonkey/Greasemonkey用ユーザースクリプトです。<br>
  Xの投稿画面から離れることなく、メディアやリンクカードを含めた投稿が可能です。

✨ 主な機能
 - シームレスなクロスポスト: Xの投稿ボタンを押すだけで、テキスト、ハッシュタグ、画像、動画、リンクカードを自動的にBlueskyへ変換して投稿します。
 - 画像: Xの現仕様に基づき、最大4枚までの同時投稿に対応。
 - 動画（.mp4形式で簡単に確認）: 動画のアップロード、自動エンコード待機機能付き。

📥 インストール方法
1. このスクリプトを実行するには、以下のいずれかのブラウザ拡張機能が必要です。Mac/iPad/iPhone の Safari と Tampermonkey の組合せでテストしましたが、他でも動くかもしれません。
 - Tampermonkey（推奨）
 - Violentmonkey
 - Greasemonkey
 - Stay など

3. スクリプトのインストール: GitHubリポジトリから最新版のインストールリンクをクリックしてください。

⚙️ 設定手順
1. X (旧 Twitter) のタイムラインを開きます。
2. 画面左側のナビゲーションメニューに追加された Bluesky設定アイコン（青い菱形のアイコン）をクリックします。
3. 表示されたパネルに以下の情報を入力し、「Save」をクリックします。
 - PDS URL: ⁠https://bsky.social⁠ (基本はそのままでOK)
 - Handle: あなたのBlueskyハンドル (⁠xxxx.bsky.social⁠)
 - App Password: Blueskyの「設定」>「アプリパスワード」で生成したもの

📝 使い方
1. Xの投稿編集画面を開きます。
2. ツールバーに追加された「Bluesky」チェックボックスにチェックを入れます。
3. 通常通りXへ投稿ボタンを押すと、自動的にBluesky側への処理が開始されます。
4. 処理の状況が投稿ボタン上に表示され、完了すると通知が表示されます。

 - リンクカード生成時にそのサイトへの接続許可を求める場合がありますので、適切な範囲で許可してください。
 - うまく動かない時は Tampermonkey を上級者設定にして、セキュリティの @connect 文を確認 項目を緩めたり、実験的項目の厳格モードを無効にしてみてください。

⚠️ 注意事項
 - セキュリティのため、App Passwordは必ずBluesky公式設定画面で生成した専用のものを使用してください。
 - 動画投稿について: 動画は ⁠.mp4⁠ 形式のみで簡単な動作確認を行っております。他のフォーマットでの動作は保証いたしかねます。
 - 動作は無保証です。予期せぬ挙動を避けるため、重要な投稿の際はバックアップを取るなどして、自己責任でご利用ください。
 - 本スクリプトはチャットAIを活用して作成されており、バグ報告や機能要望への対応にはお時間をいただく場合があります。
