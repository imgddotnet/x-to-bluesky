X（旧 Twitter ）の投稿内容を Bluesky へクロスポストする UserScript です。
素人がチャットAI（主にGemini）の力を借りて作ったのでまともに動かないと思います。
一応、注意事項は以下。いろいろすみません：

- 動作は無保証、使う場合は自己責任でお願いします
- Mac/iPad/iPhone の Safari と Tampermonkey機能拡張でテストしましたが、Tampermonkey の @connect 処理判定を緩めたりしないと動かないような気がします（互換機能拡張の Stay なら動くかも）
- 元ネタの https://github.com/59de44955ebd/twitter-to-bsky と比べると、Mastodon 処理がオミットされ、投稿内容のハッシュタグと URL がハイパーリンク化されます。リンクカードは生成できなんだ
- 画像を添付すると X にはポストされますが Bluesky にはポストされません。UserScript の限界かも
- 動画の添付はテストしてません
- GitHub の作法を知らないので、要望やリクエストをもらっても反映方法や修正の仕方すら分かりません

使い方

- Safari に Tampermonkey機能拡張を組み込んで有効化、必要に応じて設定を緩める（重要だけどよく分かってない）
- この UserScript を Tampermonkey　に読み込ませる
- X の投稿画面を表示
- 青い菱形アイコンをクリックして、あらかじめ取得しておいた Bluesky のハンドルとアプリパスワードを入力して save
- X の投稿画面にある Bluesky チェックボックスを有効にすると、投稿内容が Bluesky にクロスポストされます
- その他の隠し機能（？）は頑張ったけど期待通りの動作にならなかったです、すみません
